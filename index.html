Installation

Install Emacs

Install the latest version of Emacs via the official Ubuntu Emacs daily snapshot PPA.

    sudo add-apt-repository ppa:ubuntu-elisp/ppa
    sudo apt-get update
    sudo apt-get install emacs-snapshot

Get a local copy

Make sure you've already backed up your own configuration. Then git clone the repository.

    git clone https://github.com/izhaoyun/emacs.d.config.git ~/.emacs.d
